# צבי נינג'ה: הרפתקה בביוב

משחק הרפתקאות מגע לאייפד — לגיל 4+. **לא** תפיסת פיצות נופלות — הרפתקה אמיתית עם מפה, שלבים, אויבים, בוסים וכוחות על.

## לשחק עכשיו

**קישור:** https://guyzom.github.io/desktop-tutorial/

Safari באייפד → **שתף → הוסף למסך הבית**.

**גיבוי:**  
https://cdn.jsdelivr.net/gh/guyzom/desktop-tutorial@main/pizza-ninjas/index.html

## איך משחקים

1. בחרו צב נינג'ה (לכל אחד כוח על שונה).
2. במפת הביוב בחרו שלב פתוח.
3. **גררו למעלה/למטה** בין 3 מסלולים — הצב רץ קדימה לבד.
4. נגעו בברוקולי כדי לנצח אותו, אספו פיצות בדרך.
5. מלאו את כפתור **כוח על** ולחצו לפיצוץ מטורף!
6. כל עולם מסתיים בבוס ברוקולי ענק.

## מה יש במשחק

- 3 עולמות × 4 שלבים (כולל בוסים)
- 4 צבים עם כוחות: סערת חרבות / רעש אדמה / מגן חכם / גשם פיצות
- אויבים: שומר, מפציץ, נינג'ה + בוסים
- אלבום מדבקות + שמירה אוטומטית
- שליטה באצבע אחת, סלחני לגיל 4 (בלי "הפסדתם")

## אם הקישור לא עובד
