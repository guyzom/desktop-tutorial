# צבי נינג'ה 3D — הרפתקה בביוב

משחק הרפתקאות **תלת־מימד אמיתי** (WebGL / Three.js) לאייפד · לגיל 4+.

הדמויות, האויבים, הפיצות והביוב הם **מודלים תלת־מימדיים פרוצדורליים** — לא תמונות ספרייט.

## לשחק

https://guyzom.github.io/desktop-tutorial/

## איך משחקים

1. בחרו צב (לכל אחד נשק משלו במודל התלת־מימדי).
2. במפה בחרו שלב.
3. הצב רץ קדימה בביוב תלת־מימדי בין 3 מסלולים.
4. נגעו בברוקולי כדי לנצח, אספו פיצות, קפצו מעל סכנה.
5. כוח על כשמוכן.

## טכני

- Three.js r160 (vendored ב־`vendor/three.min.js`)
- `meshes.js` — מודלים פרוצדורליים + אנימציות + VFX
- `game.js` — מנוע הרפתקה + מגע + קפיצה
- שמירה ב־`localStorage` (`tmnt-3d-save`)

פתיחה מקומית: פתחו עם שרת מקומי.

## שליטה
- **הזזה ימינה/שמאלה** — מעקב אחרי האצבע בין מסלולים (בשחרור נצמד למסלול הקרוב).
- **הקשה קצרה** — קפיצה (אפשר לדלג מעל ברוקולי נמוך).
