# צבי נינג'ה — הרפתקת פיצה 🐢🍕

משחק הרפתקאות **תלת־מימד אמיתי** (WebGL / Three.js) לאייפד ואייפון · לגיל 4+.

הדמויות, האויבים, הפיצות והעולמות הם **מודלים תלת־מימדיים פרוצדורליים** — לא תמונות.
כל צב חמוד ואיקוני: עיניים גדולות מבריקות, מסכה צבעונית עם זנבות מתנופפים, שריון עם לוחות,
בטן קרם, וכלי נשק ייחודי (קטאנה / סאי / מקל בו / נונצ'אקו).

## לשחק

https://guyzom.github.io/desktop-tutorial/

## איך משחקים (שליטה של אצבע אחת)

1. בוחרים צב (לכל אחד כוח־על משלו).
2. במפת המסע בוחרים שלב.
3. הצב רץ קדימה לבד — **גוררים את האצבע** ימינה/שמאלה כדי לעבור בין 3 מסלולים.
4. **הקשה קצרה** = קפיצה.
5. אוספים פיצות 🍕 (זהב = שווה יותר), וכשהכוח מלא לוחצים על כפתור **כוח על ⭐**.
6. בסוף כל עולם יש בוס ברוקולי ענק — פשוט מתיישרים מולו כדי לנצח.

## עוצב לגיל 4 — בלי תסכול

- **אי אפשר להפסיד**: פגיעה רק גורמת למעידה קטנה (הצב מתכווץ, פיצה מתפזרת, רעד מסך) — אין "המשחק נגמר".
  האנרגיה מתמלאת מעצמה, וסוף השלב תמיד מנצח. גם בוס נחשף עם רשת־ביטחון לניצחון אוטומטי כדי שאף אחד לא ייתקע.
- פידבק חיובי מתמיד: צליל וניצוצות בכל פיצה, מונה קומבו, באנרים חגיגיים ("קוואבנגה!"), וכפתורים ענקיים.
- כפתור השתקה 🔊 להורים.

## עולמות

1. **הביוב** — לבנים ירוקות, צינורות, פנסים זוהרים וריר.
2. **מטבח הפיצרייה** — רצפת שחמט אדומה־לבנה, סירים, גבינה.
3. **גגות ניו־יורק בלילה** — קו רקיע עם חלונות מוארים, ירח, ואורות מסיבה.

## מבנה טכני

- `index.html` — נקודת הכניסה: מבנה המסכים, ה־HUD וטעינת הסקריפטים.
- `style.css` — כל העיצוב: RTL, HUD וממשק ידידותי־מגע.
- `vendor/three.min.js` — Three.js r160 (מקומי).
- `audio.js` — `window.GameAudio`: מוזיקת לופ פרוצדורלית לכל עולם + אפקטים קוליים (WebAudio, בלי קבצים).
- `meshes.js` — `window.GameArt`: כל המודלים התלת־מימדיים, האנימציות והאפקטים.
- `assets/` — דיוקני הצבים המצוירים למסך "בוחרים חבר" (שאר הגרפיקה במשחק פרוצדורלית).
- `game.js` — מנוע המשחק, שליטת מגע, מצלמה דינמית, שלבים, בוסים ושמירה.
- שמירה ב־`localStorage` (`tmnt-pizza-save-v2`): כוכבים, מדבקות, צב נבחר והתקדמות.
- iPad־friendly: `pixelRatio ≤ 2`, עצירת רינדור כשהלשונית מוסתרת, וכיבוד `safe-area`.

פתיחה מקומית: הריצו שרת סטטי בתיקייה זו ופתחו את `index.html` (למשל `python3 -m http.server`).
