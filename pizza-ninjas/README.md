# צבי נינג'ה 3D — הרפתקה בביוב

משחק הרפתקאות **תלת־מימד אמיתי** (WebGL / Three.js) לאייפד · לגיל 4+.

הדמויות, האויבים, הפיצות והביוב הם **מודלים תלת־מימדיים פרוצדורליים** — לא תמונות ספרייט.

## לשחק

https://guyzom.github.io/desktop-tutorial/

## איך משחקים

1. בחרו צב (צבע המסכה = הזהות שלו בתלת־מימד).
2. במפה בחרו שלב.
3. **גררו למעלה/למטה** בין 3 מסלולים — הצב רץ קדימה בביוב תלת־מימדי.
4. נגעו בברוקולי כדי לנצח, אספו פיצות תלת־מימד.
5. כוח על כשמוכן.

## טכני

- `three@0.170` מ־CDN
- `meshes.js` — בוני מודלים (צב / ברוקולי / פיצה / ביוב)
- `game.js` — מנוע הרפתקה + תאורה + צללים
- שמירה ב־`localStorage`

פתיחה מקומית: פתחו עם שרת מקומי (בגלל Three.js).
