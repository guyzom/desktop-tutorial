# 🍕 צבי נינג'ה והפיצות!

משחק מגע מטורף לאייפד — לגיל 4+ (ובמיוחד לאחיינים שמכורים לפיצות).

## איך משחקים

1. פתחו את הקובץ [`index.html`](./index.html) ב־Safari באייפד (או בכל דפדפן).
2. לחצו **בוא נשחק!** ובחרו צב: ליאו / ראף / דוני / מיקי.
3. **גררו** את הצב ימינה־שמאלה ותפסו פיצות שנופלות מהצינורות.
4. אפשר גם **ללחוץ על פיצה** באוויר!
5. כוכבים 🌟 = פיצת זהב (+5). ברוקולי 🥦 = יאק, מאפס את הרצף.

כל כמה פיצות מתחילה **סערת פיצות** — קואבונגה!

## טיפ לאייפד

ב־Safari: **שתף → הוסף למסך הבית** — והמשחק נפתח כמו אפליקציה במסך מלא.

## קבצים

- `index.html` — מסכים ו־UI
- `style.css` — עיצוב ביוב־פיצה
- `game.js` — לוגיקת המשחק + צלילים
